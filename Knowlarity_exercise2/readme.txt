- dashboard folder contains the main project.

- Home page shows the total number of teams and total matches played so far.

- A new match is created by clicking on "Create new match".

- If any of the team is new in the match and not already present in the database then that team is added to the teams list and to the database with deafult player names.
- Only the admin is allowed to add player names to the teams.

- winning team gets +10 points.
Losing team gets -10 points.

- Top 5 ranked teams are displayed by clicking "Top ranked teams".
